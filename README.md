# T-SOJA | Site Next.js

Site vitrine Next.js (App Router) pour T-SOJA Processing Factory.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS v4
- Motion (`motion/react`)

## Lancement local

1. Installer les dépendances :
   `npm install`
2. Démarrer en développement :
   `npm run dev`
3. Build production :
   `npm run build`
4. Lancer en production :
   `npm run start`

## Déploiement Vercel

- Framework Preset : `Next.js`
- Build Command : `npm run build`
- Root Directory : racine du dépôt
- Dossier d’assets : `public/images`

Le logo est utilisé dans le header, le footer et le favicon (`src/app/icon.png` et `public/favicon.png`).
